MarineThinking website based on Gatsby
-----------------------------

@2020/03/29


### Prerequisites

- node
- yarn
- gatsby cli


### Install project dependencies

```
$ yarn
```

### Local development

```
$ gatsby develop
```

then visit:

```
http://localhost:8000
```

### Build website

```
$ npm run build
```

### Dynamic content path

```
./_data/blog
```
